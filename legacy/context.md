# [기획안] 초보자용 주식 예측 서비스: "주린이 차트"

## 1. 프로젝트 개요 (Context)

* **제품명:** 주린이 차트 (Jurinee Chart)
* **타겟 유저:** 주식 차트를 전혀 볼 줄 모르는 '생초보' 투자자
* **핵심 컨셉:** 복잡한 기술적 지표 대신, 한국의 유명한 밈인 '주린이 머리' 패턴과 같은 친근한 캐릭터 비유를 사용하여 미래 주가를 예측하고 해설해주는 서비스
* **제한 사항:** 초기 단계에서는 데이터의 안정성과 패턴의 전형성을 위해 **국내 우량주(삼성전자, SK하이닉스 등)**만을 대상으로 함

## 2. 제품의 핵심 가치 (Value Proposition)

1. **시각적 직관성:** 미래의 주가 경로를 점선이나 구름대 형태로 시각화
2. **유머러스한 해설:** "골든크로스" 대신 "기영이가 팔을 들었다"와 같은 쉬운 언어 사용
3. **심리적 가이드:** 예측이 틀렸을 때도 유머를 통해 투자 스트레스를 완화

---

## 3. POC(개념 검증) 기술 요구사항

AI 에이전트가 구현해야 할 **핵심 로직**은 다음과 같습니다.

### ① 데이터 확보 (Data Ingestion)

* **방법:** Yahoo Finance API 또는 공공 데이터 API를 사용하여 삼성전자의 최근 5~10년 일봉 데이터를 JSON 형식으로 수집
* **필요 항목:** 날짜(Date), 시가(Open), 고가(High), 저가(Low), 종가(Close), 거래량(Volume)

### ② 패턴 매칭 알고리즘 (Similarity Search)

* **로직:** 현재 시점으로부터 역으로 10~20일간의 주가 변동률() 배열을 생성
* **매칭:** 과거 전체 데이터셋에서 현재의 변동률 배열과 **피어슨 상관계수(Pearson Correlation)**가 0.85 이상인 구간들을 모두 탐색
* **예측:** 매칭된 과거 구간들의 '이후 5~10일' 데이터를 평균 내어 미래 시나리오 생성

### ③ '기영이 머리' 탐지 로직 (Feature Engineering)

* 주가 급등(왼쪽 얼굴) -> 고점에서의 3회 이상 미세 파동(머리 뾰족이) -> 하락 징후(오른쪽 팔)를 감지하는 조건문 작성

---

## 4. POC 단계별 개발 태스크 (Roadmap for AI Agent)

### [Phase 1] 데이터 및 차트 기본 구현

1. `Lightweight Charts`를 활용하여 기본 캔들 차트 렌더링
2. 사용자가 종목(삼성전자/하이닉스)을 선택하면 최근 데이터를 Fetch 하는 기능

### [Phase 2] 예측 엔진 구축 (JS 중심)

1. 가장 닮은 과거 구간을 찾는 알고리즘 작성
2. 찾은 구간을 현재 차트 끝에 점선(Ghost Chart)으로 이어 붙이기
3. 예측된 경로의 신뢰 구간(Upper/Lower Bound)을 반투명한 색 영역으로 표시

### [Phase 3] 밈(Meme) 진단 및 해설 엔진

1. 패턴 분석 결과에 따른 '밈 캐릭터' 매칭 (예: 기영이, 남산타워 등)
2. **초보자용 해설 템플릿 엔진:**
* *상승형:* "기영이가 머리카락을 꼿꼿이 세웠네요! 기세가 좋습니다."
* *하락형:* "기영이가 눈물을 흘리며 퇴근하고 있어요. 잠시 관망할까요?"

## 5. 성공적인 POC를 위한 질문 리스트 (Self-Check)

* **질문 1:** 과거 패턴 5개를 섞어서 평균을 낼 것인가, 아니면 가장 닮은 1개만 보여줄 것인가? (추천: 3~5개의 평균 구름대)
* **질문 2:** '기영이' 캐릭터 이미지를 어떤 시점에 노출할 것인가? (추천: 차트의 특정 고점이나 저점에 아이콘으로 표시)
* **질문 3:** 에러 핸들링은 어떻게 할 것인가? (추천: 데이터가 없는 경우 "기영이가 낮잠 자는 중이라 데이터를 못 찾았어요"라고 표시)
