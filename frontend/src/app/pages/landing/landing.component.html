HOW IT WORKS

데이터가 증명하는 3단계 분석

01
📊

패턴 추출

현재 차트의 캔들과 지표를 정밀한 시계열 벡터 데이터(Time-Series Vector)로 변환합니다.

02
🔍

유사 패턴 매칭

Pearson·Spearman 상관분석과 DTW 알고리즘으로 5년 빅데이터에서 유사 패턴을 탐색합니다.

03
☁️

시나리오 시각화

과거의 흐름을 바탕으로 신뢰 구간(68%/95% Cloud Band)이 포함된 시나리오를 제시합니다.

TECHNICAL DEEP DIVE

주린이 차트가 데이터를 다루는 법

📊

Hybrid Correlation

단순한 가격 흐름(Pearson)뿐만 아니라 추세의 순위(Spearman)와 거래량 상관관계까지 결합하여, 인간의 눈보다 정밀하게 패턴을 대조합니다.

📐

DTW Matching

Dynamic Time Warping 기술을 적용하여, 속도는 다르지만 모양이 유사한 '시간 왜곡' 패턴까지 잡아냅니다.

🌊

Volatility Normalization

ATR(Average True Range) 기반 정규화로 종목마다 다른 변동성을 표준화하여 객관적인 비교를 수행합니다.

☁️

Probabilistic Cloud

단 하나의 선이 아닌, 통계적 확률 분포를 기반으로 한 신뢰 구름(Cloud Band)을 통해 리스크의 범위를 시각화합니다.
